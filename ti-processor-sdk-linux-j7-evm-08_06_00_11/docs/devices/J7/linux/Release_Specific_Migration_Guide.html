2.5. Processor SDK Linux Migration Guide¶

Overview

This page covers migration information for applications built on top of the Processor SDK for Linux.

2.5.1. Processor SDK 08.xx Releases¶

2.5.1.1. Processor SDK Jacinto 08.06¶

  • This is the ninth release on 2021 LTS 5.10 kernel, 2021.01 U-Boot
  • ATF 2.8
  • OPTEE 3.20
  • Yocto Dunfell/3.1
  • Platforms Supported : J721E, J7200, J721S2, J784S4, AM68A, AM69A

2.5.1.2. Processor SDK Jacinto 08.05¶

  • This is the seventh release on 2021 LTS 5.10 kernel, 2021.01 U-Boot
  • ATF 2.7
  • OPTEE 3.19
  • Yocto Dunfell/3.1
  • Platforms Supported : J721E, J7200, J721S2, J784S4

2.5.1.3. Processor SDK Jacinto 08.04¶

  • This is the fifth release on 2021 LTS 5.10 kernel, 2021.01 U-Boot
  • ATF 2.6
  • OPTEE 3.12
  • Yocto Dunfell/3.1
  • Platforms Supported : J721E, J7200, J721S2

2.5.1.4. Processor SDK 08.02¶

  • This is the third release on 2021 LTS 5.10 kernel, 2021.01 U-Boot
  • Third release on 5.10 kernel and 2021.01 U-boot
  • ATF 2.5
  • OPTEE 3.12
  • Yocto Dunfell/3.1
  • Platforms Supported : J721E, J721S2, J7200

2.5.1.5. Processor SDK 08.01¶

  • This is the second release on 2021 LTS 5.10 kernel, 2021.01 U-Boot
  • Second release on 5.10 kernel and 2021.01 U-boot
  • ATF 2.5
  • OPTEE 3.12
  • Yocto Dunfell/3.1
  • Platforms Supported : J721E, J7200

2.5.1.6. Processor SDK 08.00¶

  • This is the first release in the 2021 LTS stream, with new versions of Kernel, U-boot, toolchain and Yocto branch.`
  • First release on 5.10 kernel and 2021.01 U-boot
  • ATF 2.5
  • OPTEE 3.12
  • Yocto Dunfell/3.1
  • Platforms Supported : J721E, J7200
